#  LineBot-溫度偵測機器人

## 介紹

### 他可以做甚麼？

使用Line詢問機器人當地溫度，後機器人會回復你當地的溫度及濕度

###  需要什麼設備？

> 1. DHT11
> 2. 麵包板
> 3. ESP8266
> 4. PC
> 5. 杜邦線
    
### 需要什麼軟體？
> 1. [Python3] [Python3] 亦可使用整合環境如 [Anaconda][Anaconda]
> 2. [ArduinoIDE][ArduinoIDE]

#### Python所需Module
> [Flask][Flask] 輕便型的Python Web Server模組

    pip install flask
> [PIL][PIL] (Python Imaging Library)，Python圖像處理程式庫

    pip install pillow
